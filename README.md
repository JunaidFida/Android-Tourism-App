## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`EXPO_PUBLIC_MAPBOX_ACCESS_TOKEN`

`EXPO_PUBLIC_API_BASE_URL`

`EXPO_PUBLIC_DEV_API_HOST`

`EXPO_PUBLIC_DEV_API_PORT`

## Run Locally

Clone the project

```bash
  git clone https://github.com/JunaidFida/Android-Tourism-App.git
```

Go to the project directory

```bash
  cd TourismApp
```
Create Virtual Environment (venv)
```bash
python -m venv venv
```
Activate the venv
```bash
venv\Scripts\activate
```
Go to the directory
```bash
  cd tourist_app
```
Install dependencies

```bash
  npm install
```
Start the server

```bash
  npm start
```
Go to the directory
```bash
  cd touristapp_backend
```
Install dependencies

```bash
  pip install -r requirements.txt
```
Start the backend

```bash
  python run.py
```
